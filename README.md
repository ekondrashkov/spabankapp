## Установка и запуск проекта
1. Запустите сервер из папки /backend (выполните `npm i` для установки и `npm start` для запуска сервера).
2. Для сборки фронтэнд выполните из папки /frontend `npm i` для установки и `npm run dev` для запуска проекта в dev-режиме. Для сборки в prod-режиме выполните `npm run build`.
3. Подробные README по backend и frontend находятся в соответствиющих папках
